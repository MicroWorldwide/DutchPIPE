BETA

php/ajax avatar werelden

Met DutchPIPE open source software kunnen webdevelopers virtuele multi-user omgevingen maken. Elke webpagina wordt een abstracte omgeving of lokatie waar bezoekers en andere objecten op de pagina worden gevisualiseerd. Bezoekers kunnen door de site lopen terwijl de status van pagina's wordt behouden. Er zijn veel mogelijkheden voor real-time interactie. Het resultaat: "Persistent Interactive Page Environments." DutchPIPE gebruikt AJAX en het DOM voor de browser - het werkt zonder Java, Flash, plug-in of firewall-aanpassingen. Op de server wordt PHP 5 gebruikt.
