void setFull( [string $isFull = TRUE]   )

Fill or empty the drink

Adjusts the drink's appearance and behaviour. Without an argument, fills the drink. Drink objects are filled by default after being created.

Drink objects which are filled have the property "is_full" set to TRUE in them.


PARAMETERS: string   $isFull  TRUE to fill, FALSE to empty drink
